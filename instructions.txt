To use app:
 - cd BackEnd
 - python3 -m venv env
 - source env/bin/activate
 - pip install -U flask-cors
 - pip install -r requirements.txt
 - vim connectionString.txt
 - Enter string (You will need to request this string from team member)
 - pip install openai
 - npm run start-backend
 -
 - Now in a new terminal (keep other one open too)
 - cd ../front-end
 - brew install unixodbc
 - npm run start

 Location: 
 - actual file using flaks is Backend/base.py
 - typescript react code: front-end/src/App.tsx
